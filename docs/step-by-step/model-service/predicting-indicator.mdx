---
title: 'Predicting an Indicator'
---

```typescript
predictIndicator(id: string, klaster: string, indicator: ModelIndicator): Promise<Prediction>
```

Predicts an indicator based on the provided data.

#### Parameters

- `id` (`string`): The company identifier.
- `klaster` (`string`): The klaster to use.
- `indicator` (`ModelIndicator`): The indicator to predict.

#### Returns

A `Promise` that resolves to the predicted values (`Prediction`).

### `predictionResponse(id: string, klaster: string): Promise<PredictionResponse>`

Retrieves the prediction response for the given company data and its klaster.

#### Parameters

- `id` (`string`): The company identifier.
- `klaster` (`string`): The klaster to use.

#### Returns

A `Promise` that resolves to the prediction response (`PredictionResponse`).

#### Description

- **Initialize Prediction Response**: Create a new `PredictionResponse` object to store the prediction results.
- **Predict Financial Indicators**: For each financial indicator (Liquidity, Profitability, Efficiency, Structure, Growth):
  - Call `predictIndicator` with the company's `id`, `klaster`, and the specific indicator.
  - Receive a prediction object containing low, medium, and high values for the indicator.
  - Assign these values to the corresponding fields in the `PredictionResponse` object.
- **Return Prediction Response**: Return the populated `PredictionResponse` object.

## Types

### `ModelIndicator`

An enumeration of the different types of indicators that can be predicted.

### `Prediction`

An object containing the low, medium, and high predictions for an indicator.

### `PredictionResponse`

An object containing the predictions for multiple indicators, structured as follows:

- `model1y1`, `model1y2`, `model1y3`: Predictions for liquidity.
- `model2y1`, `model2y2`, `model2y3`: Predictions for profitability.
- `model3y1`, `model3y2`, `model3y3`: Predictions for efficiency.
- `model4y1`, `model4y2`, `model4y3`: Predictions for structure.
- `model5y1`, `model5y2`, `model5y3`: Predictions for growth.
